# pdf-box-remover
## Setup

```bash
$ pip install -r requirements.txt
```

## Usage

Modify the values of `filename` and `info` in `manual.py` as follows. However, in the following example, the file name is set to `foo`. Note that the extension `.pdf` should be omitted.

```python
# ...
filename = "foo"
info = [["index", "# of boxes"]]
# ...
```
